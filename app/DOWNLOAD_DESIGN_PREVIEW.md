## Technical Implementation

### Canvas Setup
```typescript
const canvas = document.createElement('canvas');
canvas.width = 1080;
canvas.height = 1920;
const ctx = canvas.getContext('2d');
```

### Gradient Background
```typescript
const gradient = ctx.createLinearGradient(0, 0, 1080, 1920);
gradient.addColorStop(0, '#667eea');
gradient.addColorStop(0.5, '#764ba2');
gradient.addColorStop(1, '#f093fb');
ctx.fillStyle = gradient;
ctx.fillRect(0, 0, 1080, 1920);
```

### Decorative Circles
```typescript
ctx.globalAlpha = 0.1;
circles.forEach(circle => {
  ctx.beginPath();
  ctx.arc(circle.x, circle.y, circle.r, 0, Math.PI * 2);
  ctx.fillStyle = 'white';
  ctx.fill();
});
ctx.globalAlpha = 1.0;
```

### Card Frame with Shadow
```typescript
ctx.shadowColor = 'rgba(0, 0, 0, 0.3)';
ctx.shadowBlur = 50;
ctx.shadowOffsetY = 25;
ctx.fillStyle = 'white';
roundRect(ctx, 80, 120, 920, 1680, 30);
ctx.fill();
```

### Text with Language Fonts
```typescript
const fontFamily = getFontFamily(language);
ctx.font = `36px ${fontFamily}`;
ctx.fillStyle = '#2c3e50';
ctx.textAlign = 'center';
wrapText(ctx, text, 800).forEach((line, i) => {
  ctx.fillText(line, 540, y + i * 52);
});
```

## File Output

### Specifications
- **Format**: PNG
- **Size**: ~500KB - 1MB
- **Resolution**: 1080 x 1920 pixels
- **Quality**: High (no compression)
- **Filename**: `shayari-{id}-{timestamp}.png`

### Download Trigger
```typescript
canvas.toBlob((blob) => {
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `shayari-${id}-${Date.now()}.png`;
  a.click();
  URL.revokeObjectURL(url);
}, 'image/png');
```
